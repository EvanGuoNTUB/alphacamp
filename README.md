## FizzBuzzTest
A simple program for testing FizzBuzz by Mocha and Chai test.
### Features
* 被3整除得到 Fizz
* 被5整除得到 Buzz
* 被3和5整除得到 FizzBuzz
* 若非以上狀況，則傳回輸入的數字
